The administrative module is designed for a hospital owner or the person accountable for operating the entire hospital smoothly. Using this module, you can manage the entire system easily and efficiently. It includes every important feature like doctors portal, patient portal, appointment scheduling, prescription management, human resource management, bed management, employee management, accounts, software settings, and role assigning.

The Doctor Module is included because a doctor is the most important person to ensure the health condition of any patient. Using this module, doctors can access all important functions that belong to them. This allows them to maintain patient portals, appointment scheduling & maintaining, prescription making, and any other things that are related to hospital activities.

The software with this module helps to provide a complete clinic management system that is understandable to both technical or non-technical people.

If you have an active software license, then you will have access to software support. The source also mentions this is totally open-source software.
